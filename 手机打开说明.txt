手机直接打开方法

结论

手机要直接打开并使用摄像头，请使用 HTTPS 网址，例如：

https://你的域名/你的项目/

不要把手机长期使用建立在 http://电脑IP:端口/ 上。手机浏览器通常会阻止普通 HTTP 页面调用摄像头。

最简单打开流程

1. 按“部署说明.txt”把项目发布到 GitHub Pages、Netlify、Vercel 或 Cloudflare Pages。
2. 部署完成后，复制平台给你的 https:// 地址。
3. 在手机浏览器中直接打开这个 HTTPS 地址。
4. 点击“启动摄像头”。
5. 浏览器弹出摄像头权限时，选择“允许”。
6. 伸开左右两只手，即可看到三块 AR 滤镜面片。

推荐手机浏览器

- iPhone：Safari，或 iOS Chrome / Edge。
- Android：Chrome、Edge、三星浏览器。

如果你在微信、QQ、抖音、微博等 App 内置浏览器里打开，摄像头或模型可能被限制。遇到这种情况：

1. 点击右上角菜单。
2. 选择“在浏览器打开”。
3. 用 Safari、Chrome、Edge 或三星浏览器打开同一个 HTTPS 地址。

添加到手机桌面

iPhone：

1. 用 Safari 打开 HTTPS 地址。
2. 点击分享按钮。
3. 选择“添加到主屏幕”。

Android：

1. 用 Chrome 或 Edge 打开 HTTPS 地址。
2. 点击浏览器菜单。
3. 选择“安装应用”或“添加到主屏幕”。

以后打开方式

添加到主屏幕后，直接点击手机桌面的“双手AR”图标即可打开。首次模型加载仍需要网络；加载过后页面外壳会被缓存，下次打开更快。
